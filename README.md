# CLDR Viewer

Visualize Unicode CLDR locale data completeness per language.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install cldr-viewer
```

### Fedora/RPM
```bash
sudo dnf install cldr-viewer
```
